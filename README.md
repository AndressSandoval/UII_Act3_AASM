# myapp

# Mi prompt
Has dicho
Drawer ( con 4 opciones)

encabezado: imagen avatar del negocio, desde loa red, nombre empresa, dirección, teléfono, correo,

Listile: icono, texto y acción para abrir  pagina según opción de listile, (productos, empleados, mascotas...)

En cada pagina una imagen centrada de 200x200 desde la red  (desde github)

Utilizar rutas nombradas para navegar entre paginas desde el drawer (rutas nombradas.

trabajar con firebase studio

En readme escribir el prompt para generar la app en un solo archivo segun lo anterior + las imagenes correspondientes,

probar en dartpad antes de trabajar en firbase studio.

Todo esto debe estar basado en una tienda de instrumentos, llamada "Musicman", con seccion agregada de guitarras en la barra de navegacion en main


Has dicho
dame el codigo dividido por cada pagina, el main, y el resto estara en una carpeta llamada paginas, y cada archivo estara nombrado "pagina1.dart", "pagina2.dart", etc
